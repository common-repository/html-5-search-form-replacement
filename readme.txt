A plugin to change your search form into a HTML 5 search form with placeholders.

== Description ==

Your Twenty Ten Theme is HTML 5 so your search form should be too. This plugin will change your search form into a HTML 5 search form with placeholders. Older browsers will just fall back to a normal input box.


== Installation ==

1. Upload `HTML 5 Search Form` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
